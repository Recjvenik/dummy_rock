How It Works

Up and running in under 10 minutes

No IT team needed. No app downloads. Just a browser and a join code.

1

School Signs Up

Fill the inquiry form below. We set up your school's account within 24 hours and send login credentials to the principal or IT coordinator.

2

Students Join via Code

Teacher shares an 8-character join code (e.g. AILAB8C3) on WhatsApp or classroom board. Students visit the link and are instantly enrolled.

3

Teacher Tracks Progress

Real-time dashboard shows XP earned, modules completed, streaks, and a class leaderboard. Export CSV reports for PTMs in one click.
